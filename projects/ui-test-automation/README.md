# Project: UI Test Automation

## Goal
Automate stable UI paths with maintainable page object/components.

## Suggested Stack
- Playwright + TypeScript
- Visual snapshots for critical pages
- Trace/video collection for failures

## Suggested Scenarios
- Login, onboarding, checkout/payment, logout
- Access control based on roles
- Basic accessibility checks for critical forms

```text

├── tests/
```
