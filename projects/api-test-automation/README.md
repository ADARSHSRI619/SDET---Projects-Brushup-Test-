# Project: API Test Automation

## Goal
Validate business-critical APIs with reliable, fast checks.

## Suggested Stack
- Rest Assured + TestNG (Java) or Playwright API (TypeScript)
- JSON schema validation
- Test data builders and environment configs

## Suggested Scenarios
- Auth and token lifecycle
- CRUD flows for primary entities
- Negative validation and boundary checks
- Idempotency and retry behavior
