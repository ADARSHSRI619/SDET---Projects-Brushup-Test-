# Release Readiness Checklist

## Functional
- [ ] Critical path smoke tests pass
- [ ] No open Sev-1/Sev-2 defects
- [ ] Regression run complete with approved exceptions

## Non-Functional
- [ ] Performance sanity baseline met (p95 under threshold)
- [ ] No new high-risk security findings
- [ ] Error budgets remain within target

## Operations
- [ ] Rollback plan documented
- [ ] Monitoring dashboards verified
- [ ] On-call handoff shared

## Sign-Off
- [ ] QA sign-off
- [ ] Product sign-off
- [ ] Engineering sign-off
