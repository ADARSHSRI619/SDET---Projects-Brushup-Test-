# 90-Day QA Upskilling Plan

## Days 1–30: Foundation
- Build API automation skeleton with auth, fixtures, and assertions.
- Create 15 high-value smoke scenarios.
- Set up CI trigger for pull requests.

## Days 31–60: Expansion
- Add UI regression flow for top 3 business journeys.
- Add contract tests for two key downstream services.
- Introduce flakiness tracking and root cause tags.

## Days 61–90: Maturity
- Implement performance sanity checks in pipeline.
- Add release readiness checklist automation gates.
- Publish monthly quality scorecard in repository wiki/docs.

## Deliverables
- Test strategy document
- Executable suites with reports
- Metrics dashboard screenshots
- Demo video walkthrough (5–8 minutes)
